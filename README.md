# 일정타그램 Tasktagram 

<br />

일정타그램은 Kanban 보드 스타일의 협업 관리 웹 애플리케이션으로, '할 일', '진행 중', '완료' 단계로 팀의 작업 흐름을 시각적으로 파악할 수 있습니다. 팀원들은 Instagram 스토리에서 영감을 받은 소셜 기능을 통해 각 팀원들이 현재 진행 중인 작업이 있는지를 한 눈에 확인할 수 있으며, 이를 통해 팀원 간의 효율적인 소통과 협업을 지원합니다.

<br />
<br />
